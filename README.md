# VM-Clustering
This project utilizes K-means Clustering and GMM (unsupervised learning algorithms) to cluster Virtual Machines based on their runtime behavior. 

The data for this project is pubicly available, provided by Microsoft, and represents benchmark tests across ~43,000 Azure VMs. See data here: https://github.com/Azure/AzurePublicDataset/tree/master/vm-noise-data
